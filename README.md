# Boosting Algorithms Explained: AdaBoost vs Gradient Boosting

## Overview

This tutorial demonstrates how boosting algorithms work by comparing AdaBoost and Gradient Boosting using the Titanic passenger dataset.

The objective of the tutorial is to illustrate how ensemble models improve prediction performance by combining multiple weak learners.

## Dataset

The Titanic dataset contains passenger information such as:

- passenger class
- age
- gender
- fare
- family relationships
- embarkation port

Target variable: **Survived**

Dataset source:
https://www.kaggle.com/c/titanic

## Tutorial Content

The notebook demonstrates:

- Data preprocessing
- AdaBoost model training
- Gradient Boosting model training
- Model accuracy comparison
- Confusion matrix analysis
- Learning curve visualisation
- Decision boundary interpretation
